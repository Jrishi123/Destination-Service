## 🚀 Deploy to GitHub Pages

1. Create a new GitHub repository (e.g. `destination-services`)
2. Upload all files keeping the folder structure above
3. Go to **Settings → Pages → Source → Branch: main → / (root)**
4. Your site goes live at: `https://yourusername.github.io/destination-services`
